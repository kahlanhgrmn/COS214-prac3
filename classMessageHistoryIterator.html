Detailed Description

ConcreteIterator implements for traversing the message history for the different chats.

This class provides iteration capabilities over chat message collections

Construct a MessageHistoryIterator.

Parameters
messageHistThe messages collection to iterate over
reverseIf true, iterate from the newest to oldest messages

Create copy of the message history and then sets the starting position depending on if you want to traverse from the front or the back. Forward will start at 0, and reverse will start at the last message.

Gets the next message and then moves the iterator forward.

Returns
The next message string

Returns the message that is at the current position and then moves the iterator. The direction of the movement will depend on the reverseOrder variable. Forward will increment the index and reverse will decrement the index. Will return the empty string if there are no more messages.

Implements Iterator< std::string >.

Reset the iterator to the beginning again.

Reset the current index to the start position based on the direction of iteration, forward is 0 and reverse is size - 1. This allows the iterator to be used for multiple passes over the collection.
