Detailed Description

Factory class that creates types of iterators for the system.

This class provides static methods to create iterators for different collections in the system without exposing the complexity of iterator.

Parameters

Iterator used to go through chat rooms

Create a ChatRoomIterator that allows you to go through all chat rooms that the given user is a part of. Users can belong to multiple chat rooms, so you can see which they are in. The caller must then delete the iterator that was returned when finished.
